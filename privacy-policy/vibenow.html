앱명: Vibenow  |  운영자: 소운  |  최종 업데이트: 2026-05-10

1. 수집하는 개인정보

Vibenow는 다음 정보를 수집합니다:

  • 서버 접속 로그: IP 주소, User-Agent, 접속 일시 (Vercel 호스팅 인프라에서 자동 수집)
  • 키워드 조회 통계: 사용자가 조회한 키워드 ID (서비스 개선 목적)

앱은 별도의 회원가입, 로그인, 또는 개인 식별 정보를 수집하지 않습니다.

2. 수집 목적

  • 서비스 안정성 모니터링 및 장애 대응
  • 인기 키워드 통계 분석을 통한 서비스 품질 개선

3. 보관 기간

서버 접속 로그는 Vercel 정책에 따라 30일간 보관 후 자동 삭제됩니다.

4. 제3자 제공

수집한 정보를 제3자에게 제공하지 않습니다.

5. 제3자 서비스

본 앱은 다음 외부 서비스를 사용합니다:

  • Vercel (미국): API 서버 호스팅. Vercel 개인정보처리방침

6. 이용자의 권리

이용자는 언제든지 수집된 정보의 열람, 정정, 삭제를 요청할 수 있습니다.
